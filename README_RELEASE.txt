# Local Table Order App 配布版 README

## これは何ですか？

Local Table Order App は、文化祭、学園祭、小規模イベント、小規模な出店などで使うことを想定した、ローカルネットワーク用のテーブル注文アプリです。

インターネット接続がない環境でも、サーバー用PCとタブレット・スマートフォンを同じネットワークにつなぐことで、注文画面と管理画面を利用できます。

## 主な機能

- テーブルごとの注文画面
- 管理画面
- 商品管理
- 売り切れ管理
- 注文受付ON/OFF
- 注文履歴の閲覧
- 提供済み履歴の詳細表示
- テーブル使用中ロック
- テーブルロック解除
- テーブル数の追加 / 削除
- アプリ表示名・アイコン変更
- 管理画面パスワード
- 管理PC専用のパスワードリセット

## 対応環境

この配布版は Windows 用です。

動作想定環境:

- Windows 10
- Windows 11

タブレット・スマートフォン側は、アプリをインストールする必要はありません。
ブラウザで注文画面のURLを開いて使用します。

## インストール方法

1. `LocalTableOrderApp_Setup.exe` をダブルクリックします。
2. 画面の案内に従ってインストールします。
3. インストール後、デスクトップまたはスタートメニューから `Local Table Order App` を起動します。

## 起動方法

1. `Local Table Order App` を起動します。
2. 黒い画面が開きます。
3. 自動で管理画面がブラウザに表示されます。
4. 黒い画面には、タブレット・スマートフォンで開くためのURLが表示されます。

例:

```text
管理画面:
http://127.0.0.1:5000/admin

タブレット用URL:
テーブル 1: http://192.168.x.x:5000/table/1
テーブル 2: http://192.168.x.x:5000/table/2
```

## タブレット・スマートフォンでの使い方

1. サーバー用PCと同じネットワークに接続します。
2. ブラウザを開きます。
3. 黒い画面に表示されたテーブル用URLを入力します。
4. 各テーブルの注文画面を開きます。

例:

```text
http://192.168.x.x:5000/table/1
```

テーブル1で使う端末は `/table/1`、テーブル2で使う端末は `/table/2` を開いてください。

## テーブル数を変更する場合

使用するテーブル数は、管理画面の「テーブル設定」から追加・削除できます。

テーブル設定を変更する前に、必ず注文受付を停止してください。

手順:

1. 管理画面を開きます。
2. `注文受付を停止する` を押します。
3. `テーブル設定` を開きます。
4. `テーブルを1つ追加する` または `最後のテーブルを削除する` を実行します。
5. 必要な設定が終わったら、管理画面に戻って注文受付を開始します。

注意:

- 削除できるのは最後のテーブルだけです。
- 使用中、待機中、未提供注文あり、カート残りありのテーブルは削除できません。
- テーブル番号の欠番を防ぐため、途中の番号だけを削除することはできません。

## モバイルホットスポットで使う場合

ノートPCのモバイルホットスポットを使う場合は、タブレット・スマートフォンを会場Wi-Fiではなく、ノートPCが出しているWi-Fiに接続してください。

手順:

1. Windowsの設定を開きます。
2. `ネットワークとインターネット` を開きます。
3. `モバイル ホットスポット` をONにします。
4. 表示されているネットワーク名とパスワードを確認します。
5. タブレット・スマートフォンをそのWi-Fiに接続します。
6. アプリの黒い画面に表示されたURLをブラウザで開きます。

接続できない場合は、以下のようなURLも試してください。

```text
http://192.168.137.1:5000/table/1
```

## ルーターで使う場合

Wi-Fiルーターを使う場合は、サーバー用PCとタブレット・スマートフォンを同じルーターに接続してください。

インターネット接続は必須ではありません。
同じルーター内で端末同士が通信できれば使用できます。

注意:

- ゲストWi-Fiは使わないでください。
- AP隔離、プライバシーセパレーター、端末間通信禁止がONの場合は通信できないことがあります。
- 可能であれば、サーバー用PCは有線LANでルーターに接続すると安定します。

## 管理画面の初期パスワード

初期パスワードは以下です。

```text
admin
```

使用前に、管理画面から変更してください。

## パスワードを忘れた場合

管理PCでログイン画面を開くと、下の方にパスワードリセット用のリンクが表示されます。

```text
パスワードを忘れた場合（管理PCのみ）
```

リセット画面では、確認のために以下を入力する必要があります。

```text
RESET
```

この操作は管理PCからのみ利用できます。

## 開店前チェック

使用前に、必ず以下を確認してください。

- サーバー用PCでアプリが起動している
- 管理画面にログインできる
- 必要なテーブル数になっている
- タブレット・スマートフォンから注文画面を開ける
- 商品名、価格、売り切れ設定が正しい
- 注文受付がONになっている
- テスト注文が管理画面に表示される
- 提供済み処理ができる
- 使用可能にする操作でテーブルが初期画面に戻る

## 注文受付を停止する場面

以下の操作を行う場合は、先に注文受付を停止してください。

- 商品管理
- テーブル設定
- アプリ表示名・アイコン変更
- 注文履歴の削除

注文受付を停止している間は、利用者側から新規注文・追加注文はできません。

## 終了方法

営業終了後は、黒い画面を閉じるとアプリが終了します。

終了前に、未提供の注文が残っていないか確認してください。
