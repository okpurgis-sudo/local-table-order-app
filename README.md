# テーブル注文アプリ

テーブル注文アプリ は、文化祭、学園祭、小規模イベント、小規模な出店などでの利用を想定した、ローカルネットワーク用のテーブル注文アプリです。

サーバー用PCでアプリを起動し、タブレットやスマートフォンからテーブル選択一覧画面にアクセスして利用します。
インターネット接続がない環境でも、サーバー用PCと注文用端末が同じネットワーク内にあれば使用できます。

## 主な特徴

* ローカルネットワーク内で動作
* インターネット接続なしでも利用可能
* テーブル選択一覧画面
* テーブルごとの注文画面
* 管理画面
* 商品管理
* 売り切れ管理
* 注文受付ON/OFF
* 注文履歴の閲覧
* 提供済み履歴の詳細表示
* テーブル使用中ロック
* テーブルロック解除
* テーブル数の追加・削除
* アプリ表示名・画面アイコン変更
* 管理画面パスワード
* 管理PC専用のパスワードリセット
* Windows向けexe化・インストーラー化に対応

## 想定している利用シーン

* 文化祭
* 学園祭
* 模擬店
* 小規模イベント
* 小規模な出店
* ローカル環境での注文受付テスト
* Python / Flask を使ったローカル注文システムの学習

## このアプリで行わないこと

このアプリは、売上分析アプリやクラウド注文サービスではありません。

以下の機能は意図的に含めていません。

* 売上集計
* 商品別集計
* CSV出力
* テーブル別売上集計
* クラウド連携
* オンライン決済
* 本格的なPOS機能

目的は、ローカルネットワーク内でシンプルに注文受付・商品管理・テーブル状態管理を行うことです。

## Windowsでの起動方法

### 1. Pythonをインストール

Python 3.10 以上を推奨します。

Pythonをインストールするときは、以下にチェックを入れてください。

```text
Add Python to PATH
```

### 2. 必要ライブラリをインストール

プロジェクトフォルダで以下を実行します。

```bat
pip install -r requirements.txt
```

### 3. アプリを起動

通常は以下を実行します。

```bat
run_windows.bat
```

または、直接起動する場合は以下を実行します。

```bat
python launcher.py
```

起動すると黒い画面が表示され、管理画面URLとテーブル選択一覧URLが表示されます。
また、起動直後にサーバー用PCのブラウザで管理画面が自動で開きます。

管理画面URLは以下です。

```text
http://127.0.0.1:5000/admin
```

`127.0.0.1` はサーバー用PC自身専用のURLです。
タブレットやスマートフォンでは使用できません。

## Linuxでの起動方法

このアプリは Python / Flask 製のため、Linux環境でも起動できます。

```bash
python3 --version
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python launcher.py
```

起動後、同じネットワーク内の端末から以下のようにアクセスできます。

```text
http://サーバーPCのIPアドレス:5000/
```

LinuxでIPアドレスを確認する例です。

```bash
hostname -I
```

## 起動時に表示されるURLについて

アプリを起動すると、黒い画面に以下のようなURLが表示されます。

```text
管理画面URL（このPC用）:
  http://127.0.0.1:5000/admin

テーブル選択一覧URL（タブレット・スマホ用）:
  http://192.168.x.x:5000/
```

管理画面URLは、サーバー用PCで管理画面を開くためのURLです。

テーブル選択一覧URLは、タブレットやスマートフォンから開くためのURLです。
タブレットやスマートフォンでは、黒い画面に表示された `http://192.168.x.x:5000/` のようなURLを開いてください。

## タブレット・スマートフォンからの使い方

1. サーバー用PCでアプリを起動します。
2. タブレット・スマートフォンをサーバー用PCと同じネットワークに接続します。
3. ブラウザで、黒い画面に表示されたテーブル選択一覧URLを開きます。
4. テーブル選択一覧から、使用するテーブル番号を選びます。
5. 注文画面が表示されます。

例：

```text
http://192.168.x.x:5000/
```

通常は、各テーブルのURLを直接入力する必要はありません。
テーブル選択一覧画面からテーブルを選んでください。

## モバイルホットスポットで使う場合

ノートPCのモバイルホットスポットを使う場合は、タブレット・スマートフォンを会場Wi-Fiではなく、ノートPCが出しているWi-Fiに接続してください。

手順：

1. Windowsの設定を開く
2. `ネットワークとインターネット` を開く
3. `モバイル ホットスポット` をONにする
4. 表示されているネットワーク名とパスワードを確認する
5. タブレット・スマートフォンをそのWi-Fiに接続する
6. アプリの黒い画面に表示されたテーブル選択一覧URLをブラウザで開く

モバイルホットスポットの場合、以下のようなURLで開けることがあります。

```text
http://192.168.137.1:5000/
```

## ルーターで使う場合

Wi-Fiルーターを使う場合は、サーバー用PCと注文用端末を同じルーターに接続してください。

インターネット接続は必須ではありません。
同じルーター内で端末同士が通信できれば使用できます。

注意点：

* ゲストWi-Fiは使わないでください。
* AP隔離、プライバシーセパレーター、端末間通信禁止がONの場合は通信できないことがあります。
* 可能であれば、サーバー用PCは有線LANでルーターに接続すると安定します。

## 管理画面

管理画面では以下を行えます。

* 注文受付ON/OFF
* 未提供注文の確認
* 注文を提供済みにする
* テーブル状態の確認
* テーブルを使用可能にする
* テーブルロック解除
* 注文履歴の閲覧
* 提供済み履歴の詳細確認
* 商品管理
* 売り切れ管理
* アプリ設定
* テーブル設定
* 管理画面パスワード変更

管理画面URL：

```text
http://127.0.0.1:5000/admin
```

## 管理画面パスワード

初期パスワードは以下です。

```text
admin
```

使用前に、管理画面から変更してください。

## パスワードを忘れた場合

管理PCでログイン画面を開くと、下の方にパスワードリセット用リンクが表示されます。

```text
パスワードを忘れた場合（管理PCのみ）
```

パスワードリセットは管理PCからのみ利用できます。

リセット時には、確認のため以下を入力する必要があります。

```text
RESET
```

## 注文受付ON/OFF

通常営業中は注文受付をONにします。

商品管理、アプリ設定、テーブル設定など、運用に影響する設定を変更する場合は、先に注文受付を停止してください。

注文受付停止中は、利用者側から新規注文・追加注文はできません。

## 商品管理

商品管理では以下を変更できます。

* 商品名
* 内容・単位・説明
* 価格
* 最大注文数
* カテゴリ
* 並び順
* おすすめ表示
* 売り切れ
* 表示 / 非表示

商品管理は、注文受付停止中のみ利用できます。

商品を追加・更新・削除した場合は、画面上に保存完了メッセージが表示されます。

## 売り切れ管理

商品管理で売り切れをONにすると、注文画面ではその商品を注文できなくなります。

## テーブルロック

テーブル画面を開いた時点で、そのテーブルは使用中になります。

別の端末で同じテーブルを開こうとすると、使用中として表示されます。

管理画面でテーブルを使用可能にすると、そのテーブルのロックが解除されます。

同じ端末・同じブラウザで複数タブを開いた場合は、同じ利用者として扱われることがあります。
実際の運用では、各テーブルごとに使用する端末を分けてください。

## テーブル設定

管理画面のテーブル設定から、テーブル数を追加・削除できます。

テーブル設定は注文受付停止中のみ利用できます。

削除できるのは最後のテーブルのみです。

例：

```text
現在 12 テーブル
→ 削除できるのはテーブル12のみ
```

途中のテーブルを削除すると番号が飛んで運用が混乱するため、途中のテーブル削除には対応していません。

また、以下の状態のテーブルは削除できません。

* 使用中
* 未提供注文がある
* カートが残っている
* ロック中
* 注文履歴に影響がある

テーブルを追加・削除した場合は、画面上に変更完了メッセージが表示されます。

## アプリ設定

アプリ設定では以下を変更できます。

* アプリ表示名
* 画面アイコン

この設定は画面上の表示に反映されます。
Windowsのexeアイコンを変更する機能ではありません。

アプリ設定は注文受付停止中のみ利用できます。

## 注文履歴

注文履歴画面では、過去の注文を確認できます。

提供済みの注文は詳細を開いて、どの商品が何点注文されたか確認できます。

## 注文履歴の削除

注文履歴を削除する場合は、注文履歴画面から行います。

削除時には、確認のため以下を入力する必要があります。

```text
削除する
```

安全のため、削除は注文受付停止中のみ実行できます。

成功すると以下にexeが作成されます。

## タブレット用URLが開けない場合

アプリ起動時の黒い画面には、テーブル選択一覧URLの候補が表示されます。

通常は、表示された `http://192.168.x.x:5000/` のようなURLをタブレットやスマートフォンで開けば使用できます。

ただし、モバイルホットスポット、別のWi-Fiルーター、有線LANとWi-Fiの併用、VPN環境などでは、表示されたURLで開けない場合があります。

その場合は、サーバー用PCでIPアドレスを確認してください。

### WindowsでIPアドレスを確認する方法

1. サーバー用PCで PowerShell またはコマンドプロンプトを開きます。
2. 以下を入力します。

```bat
ipconfig
```

3. 使用しているネットワークの `IPv4 アドレス` を確認します。

例：

```text
IPv4 アドレス . . . . . . . . . . . .: 192.168.137.1
```

4. タブレットやスマートフォンで以下のように入力します。

```text
http://確認したIPv4アドレス:5000/
```

例：

```text
http://192.168.137.1:5000/
```

### 注意

`127.0.0.1` はサーバー用PC自身だけで使えるアドレスです。
タブレットやスマートフォンでは使用できません。

タブレットやスマートフォンでは、`192.168.x.x`、`192.168.137.1`、`10.x.x.x` などのIPv4アドレスを使ってください。
